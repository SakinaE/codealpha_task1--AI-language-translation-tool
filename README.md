# AI Language Translation Tool

## Overview

The AI Language Translation Tool is a web-based application developed using Python and Streamlit that enables users to translate text between multiple languages. The application provides real-time translation, automatic language detection, text-to-speech functionality, and translation history tracking.

## Features

- Real-time text translation
- Automatic language detection
- Multi-language support
- Text-to-Speech (TTS) output
- Translation history
- User-friendly Streamlit interface

## Technologies Used

- Python
- Streamlit
- Deep Translator
- LangDetect
- gTTS (Google Text-to-Speech)

## Project Workflow

1. User enters text.
2. Language is automatically detected.
3. Text is translated into the selected target language.
4. Audio output is generated using Text-to-Speech.
5. Translation history is stored and displayed.

## Applications

- Language learning
- Travel assistance
- Educational purposes
- Multilingual communication
